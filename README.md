# Create Doc Numbers

Create Doc Numbers is an extension for generating fictitious CPF and CNPJ numbers. The extension was developed using React and various tools to facilitate development and ensure clean and well-structured code.

## Features

- Generation of fictitious CPF and CNPJ numbers
- Validation of CPF and CNPJ using the `cpf-cnpj-validator` library
- Interactive and user-friendly interface

## Technologies Used

- React
- Vite
- Tailwind CSS
- ESLint

## Requirements

- Node.js (version 14 or higher)
- npm (version 6 or higher)

## Installation

1. Clone the repository to your local environment:

```bash
git clone https://github.com/your-username/create-doc-numbers.git
```

2. Navigate to the project directory:

```bash
cd create-doc-numbers
```

3. Install the project dependencies:

```bash
npm install
```

## Available Scripts

In the project directory, you can run:

### `npm run dev`

Runs the app in development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload if you make edits.\
You will also see any lint errors in the console.

### `npm run build`

Builds the app for production to the `dist` folder.\
It correctly bundles React in production mode and optimizes the build for the best performance.

### `npm run preview`

Previews the production build locally.

### `npm run lint`

Runs ESLint to find and fix issues in the code.
